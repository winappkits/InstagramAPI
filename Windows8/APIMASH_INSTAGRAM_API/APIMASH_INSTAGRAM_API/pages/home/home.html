Instagram API

The Instagram api lets you look for photos posted via the Instagram application. Documentation for the api can be found here:http://instagram.com/developer/endpoints/. All calls require a client id, which you can get by registering with the Instagram api

  • Location Example

    Location related searches uses geolocation as part of the query. Search for locations that are identified in the Instagram data source near your current geolocation( or a geolocation of your choice ). Search for images based on a location using either the geolocation coordinates or a location id.

  • Tags Example

    Tag related searches allows for a query to find tags that include a name parameter ( for example 'snowy'), returning a list of matching tags. Search for recent media based on a tag name. Instagram does not support a "popular" or "commonly" used tag endpoint that could be used to create a tag cloud.

  • Popular Example

    Popular searches return the most popular items currently. A limit or number of items requested is not possible, instead you need to look at the nextUrl property or use the max or min ids.
